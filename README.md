## cryptlibc

A C library for basic string encryption/decryption. C main binary compiled with [Zig](https://ziglang.org/).

<br>

> Proof of concept  
> Not intended for production  

### Bindings

- Rust

### Instructions

Run Rust version

```{bash}
make rsbind
```

Compile using Zig CLI

```{bash}
make zigcc
```
